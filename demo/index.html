Clocklet

An opinionated clock-style vanilla-js timepicker.

Features

Intuitive - it looks like a clock
Mouse and touch friendly
3 clicks are sufficient to pick a time - am/pm, hour, minute
Click targets often used are large enough
No scrolling
Unlike Material Design, both hour and minute are available at the same time;
the transition between hour-entering and minute-entering is confusing and bothering
Keyboard and numpad friendly (by Lenientime)
Autocomplete - e.g. "1"->"01:00", "12"->"12:00", "1234"->"12:34"
Up and down arrow key support

Examples

Usage

<head>
  <link rel="stylesheet" href="../dist/clocklet.min.css">
</head>
<body>
  <input data-clocklet maxlength="5" value="12:34">
  <script src="../dist/clocklet.js"></script>
</body>

Options

<head>
  <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Lato|Frank+Ruhl+Libre:500">
  <link rel="stylesheet" href="../dist/clocklet.css">
  <style>
    .clocklet-1 {
      font-family: Lato, sans-serif;
    }
    .clocklet-2 {
      font-family: "Frank Ruhl Libre", serif;
      font-weight: 500;
      border-radius: 50%;
    }
    input {
      width: 100%;
    }
  </style>
</head>
<body style="margin-top: 280px;">
  <input
    data-clocklet="class-name: clocklet-1; format: h:mm AA; placement: top;"
    maxlength="8"
    placeholder="h:mm AA"
  >
  <input
    data-clocklet="class-name: clocklet-2; format: hh:mm a; alignment: right;"
    placeholder="hh:mm a"
    maxlength="8"
  >
  <script src="../dist/clocklet.js"></script>
</body>

Events

<head>
  <link rel="stylesheet" href="https://luncheon.github.io/clocklet/dist/clocklet.min.css">
</head>
<body>
  <input data-clocklet maxlength="5" value="12:34">
  <script src="https://luncheon.github.io/clocklet/dist/clocklet.js"></script>
  <script>
    document.body
      .querySelector('input')
      .addEventListener('clocklet.opening', function (event) {
        if (/iPhone/.test(navigator.userAgent)) {
          event.preventDefault();
          setTimeout(function () {
            clocklet.openWithoutEvents(event.target, event.detail.options);
            clocklet.root.style.transform =
              'scale(' + window.innerWidth / document.documentElement.clientWidth + ')';
          }, 100);
        }
      });
  </script>
</body>

Options

NameTypeDefaultDescription
classNamestring""Class name to set to the root element of the popup.
placement"top" | "bottom""bottom"Popup placement.
alignment"left" | "right""left"Popup alignment.
zIndexnumber | string""Popup z-order. If this value is an empty string, [(the input element z-index) + 1] is used.
formatstring"HH:mm"Time format (template) of the input element
