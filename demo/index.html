Clocklet

An opinionated clock-style vanilla-js timepicker.

Features

Intuitive - it looks like a clock
Mouse and touch friendly
3 clicks are sufficient to pick a time - am/pm, hour, minute
Click targets often used are large enough
No need to scroll
Unlike Material Design, both hour and minute are available at the same time;
the transition between hour-entering and minute-entering is confusing and bothering
Keyboard and numpad friendly (by Lenientime)
Autocomplete - e.g. "1"->"01:00", "12"->"12:00", "1234"->"12:34"
Up and down arrow key support

Options

NameTypeDefaultDescription
classNamestring""Class name to set to the root element of the popup.
placement"top" | "bottom""bottom"Popup placement.
alignment"left" | "right""left"Popup alignment.
zIndexnumber | string""Popup z-order. If this value is an empty string, [(the input element z-index) + 1] is used.
formatstring"HH:mm"Time format (template) of the input element

Events

TypeBubblesCancelableevent.detailsDescription

clocklet.openedtruefalse{ options: { ...resolvedOptions } }Raises after showing the clocklet popup.
clocklet.closingtruetrue{}Raises before hiding the clocklet popup.
clocklet.closedtruefalse{}Raises after hiding the clocklet popup.
inputtruefalse"clocklet"Raises after changing the input value.

Examples

Default

<input data-clocklet maxlength="5" value="12:34">

Options

<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Lato|Frank+Ruhl+Libre:500">

<style>
  .clocklet-options-1 {
    font-family: Lato, sans-serif;
    border-radius: 50%;
  }
  .clocklet-options-2 {
    font-family: "Frank Ruhl Libre", serif;
    font-weight: 500;
  }
</style>

<input
  data-clocklet="class-name: clocklet-options-1; format: h:mm AA; placement: top;"
  maxlength="8"
  placeholder="h:mm AA"
>

<input
  data-clocklet="class-name: clocklet-options-2; format: hh:mm a; alignment: right;"
  placeholder="hh:mm a"
  maxlength="8"
>

Events

<input class="clocklet-events" data-clocklet="placement: top;" maxlength="5" value="12:34">

<label><input class="clocklet-cancel-opening" type="checkbox">Cancel opening</label>
<label><input class="clocklet-cancel-closing" type="checkbox">Cancel closing</label>

<script>
  (function () {
    var inputElement = document.querySelector('.clocklet-events');

    inputElement.addEventListener('clocklet.opening', function (event) {
      console.log(event.type, event.target, event.detail.options)
      if (document.querySelector('.clocklet-cancel-opening').checked) {
        event.preventDefault();
      }
    });

    inputElement.addEventListener('clocklet.opened', function (event) {
      console.log(event.type, event.target, event.detail.options)
    });

    inputElement.addEventListener('clocklet.closing', function (event) {
      console.log(event.type, event.target)
      if (document.querySelector('.clocklet-cancel-closing').checked) {
        event.preventDefault();
      }
    });

    inputElement.addEventListener('clocklet.closed', function (event) {
      console.log(event.type, event.target)
    });
  })();
</script>
